# 会计分录异常检测系统

一个基于群聚类和机器学习的会计分录异常检测工具，帮助审计人员快速识别可疑的会计凭证。

## 功能特性

- **数据预处理**：自动识别科目、计算对方科目、生成凭证唯一值
- **群聚类**：基于业务活动群（采购、销售、薪酬、生产、长期资产、资金）对凭证进行分类
- **异常检测**：计算群距离矩阵，识别跨模块异常和模块内低频异常
- **机器学习补充**：对规则无法分类的凭证进行智能分类
- **可视化展示**：Sankey图展示科目资金流向

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run app.py
```

### 3. 使用流程

1. 上传会计凭证数据（Excel/CSV格式）
2. 配置字段映射（可选）
3. 查看数据概览和群聚类结果
4. 分析异常检测结果
5. 导出异常凭证报告

## 核心概念

### 会计分录特征
凭证涉及的所有一级科目，去重后按"、"连接。例如：
```
借：应收账款 100
借：应交税费 20
贷：主营业务收入 100
贷：应交税费-销项税 20
```
会计分录特征：`应收账款、应交税费、主营业务收入`

### 业务活动群

- **采购活动群**：原材料、半成品、库存商品、应付账款等
- **销售活动群**：应收账款、主营业务收入、发出商品等
- **薪酬活动群**：应付职工薪酬、应付工资等
- **生产活动群**：生产成本、制造费用等
- **长期资产群**：固定资产、在建工程、无形资产等
- **资金活动群**：短期借款、长期借款、财务费用等

### 异常检测算法

1. **跨模块异常**：跨越多群的凭证，根据群距离计算风险得分
2. **模块内异常**：根据特征在群内的出现频次计算风险得分
