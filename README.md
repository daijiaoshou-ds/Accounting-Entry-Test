# 会计分录异常检测系统

一个基于群聚类的会计分录异常检测工具，帮助审计人员快速识别可疑的会计凭证。

## 功能特性

- **智能字段识别**：自动检测日期、凭证号、科目、借贷方金额等字段，支持手动修正
- **数据预处理**：自动生成凭证唯一 ID、计算对方科目、提取一级科目
- **重要性水平分析**：基于帕累托法则（80/20 原则）自动计算各业务群的重要性水平，剔除小额凭证干扰
- **群聚类分析**：基于业务活动群（采购、销售、薪酬、生产、长期资产、资金）对凭证进行分类
- **异常检测**：
  - **跨模块异常**：识别跨越多群的异常凭证，根据群距离计算风险得分
  - **模块内异常**：识别群内低频特征凭证，根据出现频次计算风险得分
  - **α系数调整**：基于重要性水平动态调整小额凭证的异常得分
- **可视化展示**：
  - 会计科目资金流向图（有向图）
  - 风险等级分布统计
  - 各群聚类结果分析
- **结果导出**：支持导出异常凭证 Excel 报告

## 快速开始

### 1. 环境要求

- Python 3.8+
- 推荐创建虚拟环境

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 运行应用

```bash
streamlit run app.py
```

应用启动后会自动打开浏览器，访问 `http://localhost:8501`

### 4. 使用流程

1. **上传数据**：上传会计凭证文件（支持 Excel/CSV 格式）
2. **字段配置**：系统自动识别字段，可手动调整映射关系
3. **重要性水平配置**：设置帕累托累计百分比（默认 95%）
4. **开始分析**：点击"开始分析"按钮执行完整分析流程
5. **结果分析**：
   - 查看数据概览和群聚类结果
   - 分析异常检测结果（高风险/中风险/中低风险）
   - 查看会计科目资金流向图
6. **导出报告**：导出异常凭证 Excel 文件

## 核心概念

### 会计分录特征
凭证涉及的所有一级科目，去重后按"、"连接。例如：
```
借：应收账款 100
借：应交税费 20
贷：主营业务收入 100
贷：应交税费 - 销项税 20
```
会计分录特征：`应收账款、应交税费、主营业务收入`

### 业务活动群

| 业务群 | 典型科目 |
|--------|----------|
| **采购活动群** | 原材料、半成品、库存商品、应付账款、材料采购等 |
| **销售活动群** | 应收账款、主营业务收入、发出商品等 |
| **薪酬活动群** | 应付职工薪酬、应付工资等 |
| **生产活动群** | 生产成本、制造费用等 |
| **长期资产群** | 固定资产、在建工程、无形资产等 |
| **资金活动群** | 短期借款、长期借款、财务费用等 |

### 异常检测算法

#### 1. 跨模块异常
识别同时涉及多个业务群的凭证。计算公式：
```
风险得分 = 群距离 × 基础权重
```
群距离越大，异常风险越高。

#### 2. 模块内异常
识别在单一业务群内出现频次较低的特征。计算公式：
```
风险得分 = (1 - 特征频次 / 总凭证数) × 100
```

#### 3. 重要性水平调整（α系数）
基于帕累托法则，对小额凭证进行降权处理：
- 计算各业务群前 N% 金额的凭证的重要性水平
- 低于重要性水平的凭证，α系数 < 1，降低其异常得分
- 公式：`最终得分 = 风险得分 × α系数`

### 风险等级

| 等级 | 分数范围 | 标识 |
|------|----------|------|
| 高风险 | ≥80 分 | 🔴 |
| 中风险 | 30-80 分 | 🟠 |
| 中低风险 | 10-30 分 | 🟡 |
| 正常 | <10 分 | 🟢 |

## 配置说明

### 字段映射

系统支持自动识别以下字段：
- 日期列
- 凭证号列
- 科目列（一级科目）
- 借方金额列
- 贷方金额列
- 对方科目列（可选，系统可自动计算）
- 摘要列（可选）

### 重要性水平参数

- **帕累托累计百分比**：30%-99%（默认 95%）
  - 数值越小，剔除的凭证越多
  - 95% 表示只关注金额最大的前 95% 凭证

## 常见问题

### Q: 支持哪些文件格式？
A: 支持 `.xlsx`、`.xls`、`.csv` 格式的文件。

### Q: 数据格式有什么要求？
A: 数据需包含：日期、凭证编号、科目名称、借方金额、贷方金额等基本字段。

### Q: 如何解读异常结果？
A: 优先关注高风险（≥80 分）凭证，结合业务场景判断是否为真实异常。

### Q: 为什么有些凭证没有被分类？
A: 系统使用规则进行分类，未分类的凭证会归入"未分类"类别，可在结果中单独查看。
